📖 Опис

Oracle of Chaos — це API божевільного оракула, який передбачає твоє майбутнє з 100% впевненістю та 0% гарантією. Ідеально підходить для тих, хто хоче знати свою долю, але не хоче нести відповідальність за результат.

🚀 Запуск

Встановити залежності
npm install

Запустити сервер
npm start

Або для розробки
npm run dev
`

Сервер запуститься на http://localhost:3000

📡 Ендпойнти
1️⃣ GET /predict
Основне пророцтво за ім'ям

| Параметр | Тип | Обов'язковий | Опис |
|----------|-----|--------------|------|
| name | string | ✅ | Ім'я для пророцтва |

Приклад запиту:
`
GET /predict?name=ваше ім'я
`

Приклад відповіді:
`json

2️⃣ GET /predict/category/:category
Пророцтво за конкретною категорією

Доступні категорії:
• career — кар'єра та робота
• love — кохання та стосунки
• money — фінанси та гроші
• health — здоров'я
• tech — програмування та технології
• daily — щоденне життя
• special — спеціальні пророцтва

Приклад запиту:
`
GET /predict/category/love?name=Марія
`

Приклад відповіді:
